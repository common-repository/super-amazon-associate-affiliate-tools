Super helps Amazon associates easily embed affiliate products in their Wordpress blog or page. To learn more about Super, visit https://getsuper.co

== Description ==
Super is a free Amazon Associates Program plugin for WordPress that enables you to embed any product from Amazon in your Wordpress page or post. Amazon products are displayed using impactful widgets or ad units that help increase the value of your affiliate page and your earned commissions. More info on our site  <a href="http://getsuper.co" target="_blank">here</a>.

= How it Works =
1. Signup for <a href="https://dashboard.getsuper.co/sign-up" target="_blank">Super</a> and create a widget in Super dashboard. Select a widget type, add products from Amazon and customize style - all with a few clicks.
2. Get the one-line widget link and place it in your blog post or sidebar.
3. The widget gets displayed on your site.
3. On clicking the widget users get redirected to Amazon with your tracking ID. You earn commissions for all the sales you refer.

= Features =
1. Multiple widget types - Cards, lists, grids, polls or banners - choose from a growing list of widget types and place them anywhere on your website.
2. Customizable design - Design custom widgets that will perfectly match your website colors or choose from automatically generated color schemes optimized for conversions.
3. Self Updating - Amazon product prices, discounts, reviews are refreshed automatically using Amazon's Product Advertising API . Links always remain updated with your affiliate tracking ID. You don't have to take care about this!
4. Geolocation redirect - Foreign visitors are redirected to their local Amazon storefront so that you can monetize your international audience.
5. Hosted on Super - Widgets are hosted on Super infrastructure and loaded asynchronously ensuring no impact on your website loading.
6. Amazon compliant - Super is safe to use with the Amazon Associate program. Widgets are designed adhering to the Associate program guidelines with required disclosures built-in.
7. Analytics - See how your widgets perform with real-time statistics showing clicks and views. Optimize for engagement and higher affiliate returns.
8. Fast and scalable - Widgets are optimized to load only in a few milliseconds and can easily handle sites having millions of views.

== Installation ==
= Install =
1. Log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.
2. Search for "Super Amazon Associate Affiliate". Click "Install Now".

= Getting Started =
1. Sign up for a free account at <a href="http://getsuper.co" target="_blank">getsuper.co</a>. 
2. Create a widget inside Spreadr dashboard.
3. Get the widget link to be added to Wordpress.
4. Place the widget link in your blog post or sidebar to get the Amazon products displayed on your site.

= How does the plugin work? =
Super plugin supports the oEmbed protocol (<a href="http://oembed.com/" target="_blank">[http://oembed.com/]</a>) for allowing embedded representations of Super widget urls on third party sites. This plugin uses the wordpress function <a href="https://codex.wordpress.org/Function_Reference/wp_oembed_add_provider" target="_blank">[wp_oembed_add_provider]</a> to register Super as an oEmbed provider for your blog, allowing all users who can write posts to paste Super widget links and have them automatically replaced with interactive embed modules in the body of a post or page.
